/etc/passwd
===========
* 개요 : /etc/passwd는 시스템 액세스 권한이 있는 모든 사용자들을 추적하는데 사용
* 다음 정보가 들어 있으며, 콜론으로 분리된 파일
* * 사용자 이름
* * 암호화된 비밀번호
* * 사용자 ID 번호(UID)
* * 사용자 그룹 번호(GID)
* * 사용자 전체 이름(GECOS)
* * 사용자 홈 디렉토리
* * 로그인 쉘
